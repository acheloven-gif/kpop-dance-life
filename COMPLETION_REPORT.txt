ШАГ 1: Создание Telegram бота
──────────────────────────────
1. Открыть Telegram
2. Найти @BotFather
3. Отправить /newbot
4. Ввести имя и username
5. Скопировать токен

ШАГ 2: Установка проекта
────────────────────────
Windows:    setup.bat
macOS/Linux: bash setup.sh

(Это установит все зависимости автоматически)

ШАГ 3: Конфигурация
──────────────────
1. Отредактировать bot/.env
2. Вставить TELEGRAM_BOT_TOKEN
3. Сохранить файл

ШАГ 4: Запуск
────────────
Терминал 1: cd game && npm run dev
Терминал 2: cd bot && npm run dev

ШАГ 5: Тестирование
───────────────────
1. Открыть Telegram
2. Найти бота
3. /start
4. Нажать "🎮 Play Game"
5. Играть! 🎮
