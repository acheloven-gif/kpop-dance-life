  1️⃣  ШАГ 1: Создание Telegram бота
  ───────────────────────────────────────────────────────────────────────────
  
  📱 Откройте Telegram
  👤 Найдите @BotFather
  💬 Напишите /newbot
  📝 Ответьте на вопросы:
     • Имя: K-Cover Dance Life
     • Username: kcd_game_bot
  🔑 Скопируйте полученный токен (начинается с цифр)


  2️⃣  ШАГ 2: Установка проекта
  ───────────────────────────────────────────────────────────────────────────
  
  💻 На Windows:      setup.bat
  💻 На Mac/Linux:    bash setup.sh
  
  ⏳ Подождите установку зависимостей (1-2 минуты)


  3️⃣  ШАГ 3: Конфигурация
  ───────────────────────────────────────────────────────────────────────────
  
  ✏️  Отредактируйте файл: bot/.env
  
  ПЕРЕД:
  ┌─ TELEGRAM_BOT_TOKEN=YOUR_BOT_TOKEN_HERE
  
  ПОСЛЕ:
  ┌─ TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklmnopqrstuvwxyz
  └─ WEB_APP_URL=http://localhost:3000/game
  
  (Замените 123456789:ABCde... на ваш реальный токен от BotFather)


  4️⃣  ШАГ 4: Запуск серверов
  ───────────────────────────────────────────────────────────────────────────
  
  Откройте ДВА отдельных терминала:
  
  📍 ТЕРМИНАЛ 1 (Игра):
  ├─ cd game
  └─ npm run dev
     ✅ Игра откроется на http://localhost:5173

  📍 ТЕРМИНАЛ 2 (Бот):
  ├─ cd bot
  └─ npm run dev
     ✅ Бот запустится на http://localhost:3000


  5️⃣  ШАГ 5: Тестирование
  ───────────────────────────────────────────────────────────────────────────
  
  📱 Откройте Telegram
  🔍 Найдите вашего бота (@kcd_game_bot или как вы назвали)
  👆 Нажмите /start
  🎮 Нажмите кнопку "🎮 Play Game"
  
  ✅ Веб-приложение откроется в боте и вы можете играть!

  ❌ "Бот не отвечает"
  ───────────────────────────
  ✅ Проверьте bot/.env - правильный ли токен?
  ✅ Запущен ли сервер бота? (npm run dev в папке bot)
  ✅ Проверьте консоль на ошибки

  ❌ "Веб-приложение не загружается"
  ───────────────────────────
  ✅ Собрана ли игра? (cd game && npm run build)
  ✅ Правильный ли WEB_APP_URL в bot/.env?
  ✅ Откройте консоль (F12) и проверьте ошибки

  ❌ "Данные не сохраняются"
  ───────────────────────────
  ✅ Проверьте что localStorage включен
  ✅ В production требуется HTTPS
  ✅ Fallback работает автоматически
