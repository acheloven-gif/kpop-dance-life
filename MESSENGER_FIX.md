# Исправление системы сообщений в мессенджере

## Проблема
Ответы на предложения коллаба и заявки на вступление в команду не приходили в мессенджер.

## Решение

### 1. GameContext.tsx - Обработка ответов на предложения коллаба (строка 2545+)
- Добавлена ежедневная обработка готовых предложений (`queuedCollabProposals`)
- Когда наступает день ответа, NPC отправляет сообщение с типом `collab_response`
- Вероятность принятия зависит от количества дней, прошедших с момента предложения:
  - дни 1-5: 70% шанс
  - дни 6-10: 50% шанс
  - дни 11-15: 30% шанс
  - дни 16+: 10% шанс

### 2. GameContext.tsx - Обновлена функция proposeCollab (строка 446)
- **Критическое изменение**: Теперь при отправке предложения NPC автоматически устанавливается как 'acquaintance'
- Это делает NPC видимым в списке контактов мессенджера (критерий: `relationship !== 'stranger'`)
- Без этого игрок отправляет сообщение, но NPC не появляется в контактах

### 3. GameContext.tsx - Типизация queuedApplications (строка 315)
- Добавлено поле `attempted?: boolean` для отслеживания обработанных заявок

### 4. GameContext.tsx - sendTeamApplication (строка 1399)
- Добавлено `attempted: false` при создании заявки
- Обеспечивает правильное отслеживание статуса обработки

### 5. GameContext.tsx - Обработка team applications (строка 2509+)
- Добавлена проверка `!q.attempted` при фильтрации готовых заявок
- Добавлено обновление `attempted: true` после обработки
- Предотвращает множественную обработку одной заявки

### 6. MessengerButton.tsx - UI обработка team_application_response (строка 554+)
- Добавлен вывод статуса ответа на заявку (принята/отклонена)
- Зелёное сообщение для принятых заявок, красное для отклоненных

## Тестирование
1. Отправьте предложение коллаба → NPC должен появиться в контактах мессенджера
2. Подождите 1-7 дней игровых дней → должно прийти сообщение-ответ в мессенджер
3. Отправьте заявку в команду → должно прийти сообщение-ответ в мессенджер через 1-7 дней

## Файлы изменены
- `game/src/context/GameContext.tsx` - основная логика обработки
- `game/src/components/MessengerButton.tsx` - UI отображение ответов
